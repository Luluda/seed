# 🌸 待产 & 母婴用品清单

为 **预产期 2026-07-02 · 浙江省妇保 · 杭州** 量身整理的交互式清单。

## 在线使用

打开 GitHub Pages（在仓库 Settings → Pages → 选择 main 分支 root 启用）：
`https://luluda.github.io/seed/`

或者本地：直接双击 `index.html` 即可在浏览器打开（无需任何依赖）。

## 特性

- **按场景分组**：证件 / 产房 / 陪护 / 住院 / 月子家用（喂养·洗护·睡眠·出行·环境·药品·玩具）/ 后续添置
- **优先级标签**：必备 ⭐ / 推荐 / 可选
- **双档预算**：每项给出"基础款"与"进阶款"品牌候选 + 价格区间
- **多维筛选**：优先级 · 状态 · 顺产/剖产 · 一体化病房（隐藏院方提供项）· 关键词搜索
- **三态切换**：未购 / 已购 / 跳过（自动保存到浏览器 localStorage）
- **进度面板**：总进度、必备进度、基础/进阶预算估算
- **导入导出**：JSON 格式，可跨设备同步
- **打印友好**：一键打印，自动隐藏跳过项和按钮

## 数据结构

物品数据在 `data.js` 中，可直接编辑增删。每个条目包含：

```js
{
  id: 'unique-id',
  scene: 'home_feed',       // 所属场景
  name: '物品名',
  qty: '数量',
  priority: 'must|rec|opt',  // 优先级
  tag: ['顺','剖','夏','院内','一体化'],  // 标签
  brands: [{ name, tier:'basic|premium', price, note }],
  note: '使用经验/备注',
}
```

## 维护

- 状态保存在 `localStorage`（key: `mb-checklist-v1`），换浏览器记得先**导出 JSON**
- 修改 `data.js` 后刷新页面即可
- 想新增分类，在 `SCENES` 中加一条，`scene` 字段对应即可
